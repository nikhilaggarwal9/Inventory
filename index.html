📧 Low stock email alerts
Automatically send an email to the admin when any item's stock drops below 35% of its initial value. Uses EmailJS — a free service to send emails directly from the browser.
Setup guide (one-time)
1
Create a free account at emailjs.com — free tier allows 200 emails/month.
2
Go to Email Services → Add New Service → connect Gmail or Outlook. Copy the Service ID.
3
Go to Email Templates → Create Template. Use: {{items_list}}, {{sent_at}}, {{total_alerts}}. Copy the Template ID.
4
Go to Account → Copy your Public Key.
5
Paste all three values below along with your admin email, then click Save & Test.

🔥 Firebase setup
This app uses Firebase Firestore for real-time cross-device sync. One-time setup required:
1
Go to console.firebase.google.com → Create a project (free Spark plan works).
2
Click Firestore Database → Create database → choose Start in test mode.
3
Go to Project Settings → scroll to Your Apps → click the </> Web icon → register the app → copy the firebaseConfig object.
4
Open this HTML file in a text editor. Find const FIREBASE_CONFIG = { near the top and replace the placeholder values with your real config values. Save and re-host the file.
Data management
All data is synced to Firebase Firestore and shared across all devices in real time. Export a CSV backup regularly for your records.
